Definition

What is CSS?

CSS (Cascading Style Sheets) is the language used to control the visual presentation of HTML documents — colours, fonts, spacing, layout, animations, and responsive design. HTML provides the structure; CSS provides the appearance.

The word "cascading" describes how multiple style rules can apply to the same element. CSS resolves conflicts through a priority system called specificity, combined with source order and the concept of inheritance.
Core Concept

The Cascade & Specificity

When multiple rules target the same element, CSS calculates a specificity score to decide which wins. The scoring system uses four levels:
specificity.css — Priority from low to highCSS3
12345678910
/* Specificity from lowest to highest: */ * { color: black; } /* (0,0,0,0) Universal */ p { color: blue; } /* (0,0,0,1) Element */ .highlight { color: green; } /* (0,0,1,0) Class */ #hero { color: red; } /* (0,1,0,0) ID */ p { color: purple !important; } /* Overrides ALL */ /* Inline style beats everything except !important */ /* <p style="color:orange"> → (1,0,0,0) */ /* Inheritance: children inherit font, color, etc. */

Core Concept

The Box Model

Every HTML element is a rectangular box. The CSS box model describes the four areas around every element. Hover each layer in the diagram to learn about it:
Hover a layer on the diagram The box model has four layers: content, padding, border, and margin. Together they determine an element's total space on the page.
Set box-sizing: border-box to include padding and border inside the declared width — this is what almost all modern CSS does.
Feature

CSS Custom Properties (Variables)

Custom properties (prefixed with --) defined on :root act as global CSS variables. Change one value and it updates everywhere instantly. They can even be manipulated with JavaScript.
variables.css — Design token systemCSS3
123456789101112131415
/* Define tokens on :root for global scope */ :root { --primary: #e91e63; --font-size-base: 16px; --radius: 12px; --shadow: 0 8px 32px rgba(0,0,0,.2); } .card { border-radius: var(--radius); box-shadow: var(--shadow); border-top: 3px solid var(--primary); } /* Override locally (scoped variable) */
