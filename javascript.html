Definition

What is JavaScript?

JavaScript is a lightweight, interpreted, multi-paradigm programming language that runs natively in every web browser. Unlike HTML (structure) and CSS (style), JavaScript adds behaviour — it can respond to user events, modify the page after it loads, communicate with servers, and build entire applications.

Created in just 10 days in 1995 by Brendan Eich at Netscape, JavaScript has evolved from a simple scripting language into one of the most widely used programming languages on Earth — running in browsers, servers (Node.js), mobile apps, IoT devices, and even space systems.
History

The Evolution of JavaScript

1995
Birth of JavaScript
Brendan Eich creates Mocha (renamed LiveScript, then JavaScript) at Netscape in 10 days. Designed to let web pages respond to clicks and validate forms client-side without a server round-trip.
1997
ECMAScript Standard (ES1)
JavaScript is standardised as ECMAScript by ECMA International, enabling cross-browser compatibility and defining the language specification independently of any vendor.
ES1
2005
Ajax Revolution
Jesse James Garrett coins "Ajax" — Asynchronous JavaScript and XML. Gmail and Google Maps prove that JS can build rich, desktop-like experiences without full page reloads.
XMLHttpRequestAsync
2009
Node.js — JavaScript on the Server
Ryan Dahl releases Node.js, bringing JavaScript to the backend. For the first time, one language can power both the browser and the server — the full stack.
Node.jsV8 Engine
2015
ES6 / ES2015 — The Modern Era
The biggest update to JavaScript ever. Introduces arrow functions, classes, template literals, destructuring, Promises, let/const, modules, and more. This is the JavaScript developers write today.
Arrow FunctionsPromisesModulesClasses
2017
ES8 — async / await
async/await syntax makes asynchronous code read like synchronous code, eliminating "callback hell" and making server communication dramatically cleaner to write and understand.

Annual ECMAScript releases continue. JavaScript powers React, Vue, Angular, Next.js, React Native, Electron, and more. It's the most-used language on GitHub for over a decade running.

Core Concepts

Key Language Features

01
Variables & Types
let / const (block-scoped), var (function-scoped). Dynamic typing: strings, numbers, booleans, null, undefined, objects, arrays, symbols.
02
Functions
First-class citizens. Function declarations, expressions, arrow functions ( => ), immediately invoked (IIFE), higher-order functions (map/filter/reduce).
03
DOM Manipulation
Document Object Model — a live tree of all HTML elements. JS can read, modify, add, or remove any node: querySelector, innerHTML, classList, setAttribute.
04
Events
addEventListener() attaches handlers to any DOM element. Events: click, mouseover, keydown, submit, load, scroll, resize, input — the foundation of interactivity.
05
Promises & Async
Asynchronous patterns for handling time-delayed operations (network requests, timers). Promises, .then().catch(), and async/await for readable async code.
06
Fetch API
Native browser API for HTTP requests. fetch(url).then(r => r.json()) — replaces XMLHttpRequest. Works with REST APIs, returning Promises of response data.
